# Ansible Role: ImageMagick

An Ansible role that installs [ImageMagick](http://www.imagemagick.org/script/index.php) on RHEL/CentOS and Debian/Ubuntu.


## Requirements

None.


## Role Variables

None.


## Dependencies

None.


## Example Playbook

```
- hosts: servers
  roles:
     - { role: hashbangcode.imagemagick }
```
